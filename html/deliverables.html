This page outlines all Bootstrap components and dynamic JavaScript features implemented in my portfolio website, as required by the assessment brief.

1. Bootstrap Components Used

  • Navbar - Responsive navigation with collapsible menu and toggler
  • Container & Grid System - Rows, columns (col-md-6, col-lg-4, etc.) for responsive layout
  • Buttons - Primary, outline, small buttons (btn-primary, btn-outline-light, btn-sm)
  • Cards - Project cards, education cards, hobby cards with headers and bodies
  • Accordion - Education section with collapsible year panels
  • Modal - Contact information popup and email provider chooser
  • Progress Bars - Animated skill level indicators in Skills section
  • Badges - Tech stack labels on project cards (badge-tech-html, badge-tech-js, etc.)
  • Alerts - Info alerts for work experience fallback messages
  • List Groups - Styled lists for deliverables and work experience
  • Bootstrap Icons - Icons throughout site (bi-download, bi-controller, bi-linkedin, etc.)
  • Collapse Component - Used in navbar toggler and accordion functionality
  • Utility Classes - Spacing (m-, p-), display (d-flex, d-none), text (text-center, text-muted), colors (bg-dark, bg-light), shadows, rounded corners

2. Dynamic JavaScript Features Implemented

  • Loading Screen - Animated spinner overlay that fades out when page loads
  • Dark Mode Toggle - Theme switcher with localStorage persistence across page visits
  • Smooth Scrolling - Smooth navigation to sections when clicking navbar links
  • Project Filtering - Filter project cards by year (2nd, 3rd, 4th year or all)
  • Animated Progress Bars - Skill bars animate when scrolled into view
  • Email Provider Chooser - Modal allowing users to select their preferred email client
  • Work Experience Loader - Dynamically populates work experience from JSON data
  • Navbar Link Normalizer - Adjusts navbar links to work correctly in both root and subdirectories
  • Mobile Menu Auto-close - Automatically closes collapsed navbar after clicking a link
  • Theme Icon Sync - Updates dark mode toggle button icon based on current theme
